# Library

A Library App built as part of [The Odin Project Library Project](https://www.theodinproject.com/lessons/node-path-javascript-library) curriculum to practice object-oriented programming, DOM manipulation, and managing application state with JavaScript.

---

## Features

* Add new books to the library
* Remove books from the library
* Display book information dynamically
* Toggle book read status
* Rate book
* Store books in an array-based data structure
* Interactive form handling
* Dynamic DOM rendering

---

## Tech Stack

* JavaScript
* HTML
* CSS

---

## What I Learned

* Object constructors / classes
* Managing application data with arrays and objects
* DOM manipulation
* Form handling in JavaScript
* Event delegation
* Structuring small JavaScript applications

---

## Challenges

* Keeping UI and data synchronized
* Dynamically rendering book cards
* Managing state updates after user interactions
